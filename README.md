# Project 1: Sales Analysis
**Database:** AdventureWorks  
**Tools:** SQL Server Management Studio 22, T-SQL  

## Business Questions Answered
1. Which products generated the most revenue?
2. Which product categories drove the most revenue?
3. How are orders distributed across Small, Medium, and Large segments?
4. How has monthly revenue trended over time?

## Key Findings
- The top product by revenue was **Mountain-200 Black, 38** at $4.3M
- **Bikes** dominated category revenue at $94M
- The majority of revenue came from **Large orders** over $10,000
